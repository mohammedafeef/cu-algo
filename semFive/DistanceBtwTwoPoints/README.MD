## problem: Distance Between Two Points

### Algorithm

*Step 01* : **Start**

*Step 02* : **Declare variables x1, y1, x2, y2 and d**

*Step 03* : **Read first point as (x1, y1) and second point as (x2, y2)**

*Step 04* : **Calculate d= &#8730;( (x2-x1)<sup>2</sup> + (y2-y1)<sup>2</sup> )**

*Step 05* : **Print d**

*Step 06* : **Stop**
